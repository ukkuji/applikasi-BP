# Cilist
Simple CRUD app built with MySQL, ExpressJS, React, & Node.

`Database    : db:3306`

`Backend     : backend:5000`

`Frontend    : frontend:3000`

# Setup

You can deploy this app with just 1 command 

```
docker-compose up -d --build
```
